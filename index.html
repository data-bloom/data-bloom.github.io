Hadoop Data Governance

Governance with end-to-end Metadata Management

If you have embarked on a hadoop datalake, you are already familiar with how spotty data governance of the lake can be.

Governance is not an isolated data-dictionary, quality, business, process, or tool problem. The datalake must implicitly support automatic metadata capture, tagging, deduplication, profiling, quality, provenance, propensity, and security controls. Our solution provides a WYSIWYG data "manifold" to enforce end-to-end automated metadata controls across the full data lifecycle.

Data Curation & Linking

Automatically collect, curate, and collate raw data through your enterprise data hierarchy

Data is the new oil. But hoarding raw data without transforming it to the enterprise standard is unproductive.

Datalake is great to contain data in a single schema-less secure logical warehouse. But if your datalake is not shored up with an enterprise data taxonomy, the datasets are merely "colocated", not correlated!

Data Bloom enables unstructured fuzzy quadratic joins so the datalake can be used to curate raw data and identify latent correlations. Employ Data Bloom to continuously process raw data, mine patterns, link entities, derive business ontologies in a mixed-initiative environment.

Personalized Search Analytics

Deliver superior outcomes for your customers with personalized search

Text search yields good results. But superior results go beyond simple search by combining explicit (stated), implicit (implied), and tacit (observed) intelligence. Each user, context, content, goals are different. Offering same results for diverse personas will indispensably disappoint customers.

Demystifying user's complex knowledge goals is very challenging, but user's digital fingerprints can be handy to infer more than is apparent from a simple search query. Tracking pixel instrumentation is necessary for capturing this intelligence. Mining these homing signals of personalization must be a critical operational component of every search/knowledge enterprise.
